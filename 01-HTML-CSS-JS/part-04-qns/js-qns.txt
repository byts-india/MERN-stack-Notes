## 2Ô∏è‚É£ Functions & Scope (Very Frequently Asked)

16. What is a function?
  - function is a concept. where reusable codes are kept.
  - so that we can call it when ever we want. 

17. Types of functions in JavaScript?
  - normal function
  - function expression
  - arrow function

18. What is function hoisting?
  - keeping function declartion at bottom of file.
  - as process of moving all declaration to top by js, is called hoisting.
  - func call is still valid.

19. What is scope in JavaScript?
  - scope reference the access and availability of a variable or func 
    in a context is called scope.

20. Types of scope?
  - local or block scope
  - function scope
  - global scope

21. What is global scope?
  - a variable or func defined outside. 
  - which accessible from anywhere.

22. What is local scope?
  - it is block level scope
  eg:
    for(let i=0; i<2; i++) {
      console.log(i);
    }
  - this variable i is not access outside the for loop.
  - so variable i is local scope

23. Difference between function scope and block scope?
  - declared and accessible within the function is called function scope.
  - declared and accessible within the block is called block scope.

24. What is closure?
  - is created when a function remembers and accesses variables from
    it outer ( parent ) scope. even after the outer function has 
    finished it execution.

    function + its lexical scope = closure 

25. Why closures are used?
  - data hiding ( private variables )
  - maintain state btw functions calls
  - callbacks & async operations.
  - etc.

26. What is callback function?
  - is a function passed to another function.
  - which will be called later is called callback function
  eg:
    setTimeout(() => { console.log("called me."); }, 2000);
    here `() => {console.log("called me."); }` is a callback func.


27. What is IIFE?
  - Immediately invoked function execution

  (() => {
    console.log("hello");
  })();

  - arr.unshift()    : add elements at first / left
  - arr.shift()     : remove elements at first / left

42. Difference between `slice()` and `substring()`?
  - slice()
    - can accept neg indexs
    - more flexible
    - works with string & arrays
  - substring()
    - does not support neg indexes
    - auto swap start and end. if start > end
    - works only with string.

43. How to reverse a string?
  let arr = "abcde".split('');
  arr.reverse()
  s = arr.join("")
  console.log(s);

44. How to check palindrome?
  let val = "madam";
  n = val.length

  flag = true;

  for(let i=0; i<n/2; i++) {
      if (val[i] != val[n-i-1]) {
          flag = false;
      }
  }

  console.log(flag ? 'palindrome' : 'not a palindrome');

45. How to convert string to number?

    - parseInt('123')
46. Difference between `parseInt()` and `Number()`?

  - parseInt() stops cconversion when it hits a non numeric char
  - Number() if non numeric found, it stores NaN.

47. What is `toFixed()`?
